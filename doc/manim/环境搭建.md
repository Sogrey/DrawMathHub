# Manim 环境搭建

## 要求

| 项 | 推荐 |
|----|------|
| Python | 3.8+（项目已在 3.8 验证；类型注解需 `from __future__ import annotations`） |
| Manim | Community Edition **0.18.x**（`pip install manim`） |
| FFmpeg | 后处理切分分段视频必需 |
| 字体 | Windows：`Microsoft YaHei`（`safe_video.DEFAULT_FONT`） |

## 安装 Manim CE

```bash
pip install manim
```

验证：

```bash
python -m manim --version
```

## 环境检测（推荐）

使用个人 skill 自带脚本（若已安装 `manim-video-safe`）：

```bash
python %USERPROFILE%\.cursor\skills\manim-video-safe\scripts\check_env.py
```

检查项通常包括：Python、manim、ffmpeg、LaTeX（若用 MathTex）、中文字体。

## 常见问题

### 渲染中断后再次失败

删除对应 Scene 的缓存目录后重渲：

```
manim/scenes/{methodType}/media/videos/problem_{n}/**/partial_movie_files/
```

### MathTex / LaTeX

`problem_1.py` 作答段使用 `MathTex`；本机需安装 LaTeX 发行版。若仅调试布局，可暂时用 `safe_text` 代替公式。

### GLib-GIO 警告（Windows）

渲染时可能出现 UWP 应用相关警告，一般可忽略，不影响输出。

### Manim 版本差异

| 版本 | 注意 |
|------|------|
| 0.18 | 用 `scene.renderer.time` 取当前时间，无 `scene.time` |
| 0.20+ | 升级前需回归测试 Scene 与 API |

## 项目内模块位置

无需从 skill 复制 `safe_video.py`——项目已维护副本：

```
manim/scenes/_shared/
├── safe_video.py      # SafeScene
├── lesson_base.py     # MathLessonScene
├── video_export.py    # 分段记录 + manifest
├── post_render.py     # 渲染后归档
└── export_segments.py # ffmpeg 切分
```

脚本通过 `sys.path` 引用 `_shared`（见 `problem_1.py` 头部）。
