# Manim 脚本编写

## 文件组织

```
manim/scenes/
├── _shared/                    # 公共模块（勿在 methodType 子目录重复造轮子）
│   ├── safe_video.py
│   ├── lesson_base.py
│   └── video_export.py
└── {methodType}/               # 与 JSON methodType 一致，共 18 种
    └── problem_{lessonNumber}.py
```

| 规则 | 示例 |
|------|------|
| 文件名 | `problem_1.py` ↔ `public/data/problems/1.json` |
| 类名 | `Problem1Scene` |
| 讲次号 | 基类从文件名自动解析，无需手写 `LESSON_NUMBER` |
| 母题/举一反三 | `EXAMPLE_INDEX = 0` 为母题，1+ 为 extensionProblems 序号 |

## 最小骨架

```python
from lesson_base import MathLessonScene
from manim import *

class Problem1Scene(MathLessonScene):
    EXAMPLE_INDEX = 0

    def construct(self):
        data = self.load_problem()
        self.init_video_recorder()

        with self.segment("cover", "封面", "segments/cover.mp4", "片头", gap_after=False):
            self.show_title(data["title"], subtitle=f"画图解题法 · {data['methodType']}")

        self.init_layout_after_title(prob_h=1.0)

        # … intro / question / draw-* / written / keypoints / end …

        self.finalize_lesson()  # 写出 manifest.json
```

## 核心 API

### 布局（`safe_video.py` / `lesson_base.py`）

| API | 用途 |
|-----|------|
| `show_title()` | 片头；缩移后左上角主副标题同行 |
| `init_layout_after_title()` | 计算 prob_top、step_y、draw_y 等 |
| `place_section_title()` | 段内小标题（相对左上角标题底边） |
| `place_below_section_title()` | 段内正文紧跟小标题 |
| `make_problem_box()` | 题目框 |
| `step_label()` | 图解步骤提示行 |
| `safe_subtitle()` | 底部解释字幕（位置偏下，少占画图区） |
| `safe_wrapped_text()` | 自动折行；JSON 中 `\n` 可强制换行 |

### 之间问题图解（`make_between_diagram`）

```python
diag = self.make_between_diagram(lingling_rank, lili_rank, self.layout["draw_y"])
# 前缀圈/省略号 + 左关键人 + 之间(边圈+横省略号+边圈) + 右关键人 + 后缀圈
```

待求区域第四步标红：`diag["between_zone_circles"]` + `diag["between_ellipsis"]`。

### 关键点拨

```python
# 仅文字
self.play_keypoints_only(mp["keyPoints"], wait=6)

# 图解放大 + 点拨在下方（作答段勿淡出 diagram_all）
self.play_keypoints_only(
    mp["keyPoints"], wait=6,
    diagram=diagram_all, from_scale=0.55,
)
```

### 分段包裹

```python
with self.segment("draw-1", "1", "segments/03.mp4", "步骤说明"):
    ...
```

- `label`：画图段用 `"1"`、`"2"`；其他段用语义名（题型讲解、作答…）
- `gap_after=False`：通常用于 cover、end
- 完整版段间自动 `wait(3)`，不计入分段 mp4

## 数据读取

```python
data = self.load_problem()           # public/data/problems/{lessonNumber}.json
mp = self.main_problem               # mainProblem 字段
mp["keyPoints"]                      # 关键点拨文案
mp["originalQuestion"]               # 原题
```

`keyPoints` 过长时在 JSON 中用 `\n` 断句。
