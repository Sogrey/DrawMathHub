## 文件组织

```

├── _shared/
│   ├── safe_video.py

│   └── diagrams/               # 按题型，非按讲次
│       ├── between.py
```

| 规则 | 示例 |
|------|------|
| 文件名 | `problem_9.py` ↔ `public/data/problems/9.json` |
| 类名 | `Problem9Scene` |
| 讲次号 | 基类从文件名自动解析 |
| 母题/举一反三 | `EXAMPLE_INDEX = 0` 为母题，1+ 为 extensionProblems 序号 |
| 子目录 | 第9讲 `线段图法/`、第4讲 `列表法/`… 勿全放 `图示法/` |

## 最小骨架

```python
from lesson_base import MathLessonScene
from manim import *

class Problem9Scene(MathLessonScene):
    EXAMPLE_INDEX = 0

    def construct(self):
        data = self.load_problem()
        self.init_video_recorder()
        mp = self.main_problem

        with self.segment("intro", "题型讲解", "segments/01.mp4", "片头标题与题型讲解"):
            self.show_title(data["title"], subtitle=f"画图解题法 · {data['methodType']}")
            self.init_layout_after_title(prob_h=1.0)
            # 概念 + layout_numbered_features（3 条特征）

        with self.segment("question", "题目", "segments/02.mp4", "读题"):
            prob_all = self.make_problem_box("题目：...", "...")

        draw_y = self.layout["draw_y"]
        diag = self.make_boat_rental_diagram(34, 4, draw_y)

        # draw-1 … draw-n：每段 self.add 保留 prob_all 与已有图解部件
        # written → keypoints → end
        self.finalize_lesson()
```

## 核心 API

### 布局（`safe_video.py` / `lesson_base.py`）

| API | 用途 |
|-----|------|
| `show_title()` | 片头；缩移后左上角主副标题同行 |
| `init_layout_after_title()` | 计算 prob_top、step_y、draw_y 等 |
| `place_section_title()` | 段内小标题（相对左上角标题底边） |
| `place_below_section_title()` | 段内正文紧跟小标题 |
| `make_numbered_feature_row()` | 单条特征行（序号圆 + 主句 + 副句） |
| `layout_numbered_features()` | 特征列表：≤3 单列，≥4 左右双列 |
| `make_problem_box()` | 题目框 |
| `step_label()` | 图解步骤提示行 |
| `safe_subtitle()` | 底部解释字幕（位置偏下，少占画图区） |
| `safe_wrapped_text()` | 自动折行；JSON 中 `\n` 可强制换行 |

### 图解模块（按题型选用）

```python
# 之间问题（第1讲）
diag = self.make_between_diagram(6, 16, draw_y)

# 租船进一（第9讲）
diag = self.make_boat_rental_diagram(34, 4, draw_y)

# 锯木头（第8讲）
diag = self.make_saw_diagram(3, 4, draw_y)
```

Mixin 应返回 `dict`，含分步动画部件与 `layout_row`；详见各 `diagrams/*.py`。

### 图解段保留元素

```python
with self.segment("draw-3", "3", "segments/05.mp4", "..."):
    self.add(prob_all, diag["hint"], diag["line_block"], diag["full_block"])
    # 勿 FadeOut 前几步的辅助说明图
```

### 作答段（推荐）

```python
written_diagram_scale = 0.58
# 缩小图解到右下 → 左侧分步 FadeIn 列式
# 列式前加 GREY_B 文字说明 → 算式 → 答（YELLOW + 红框）
# 图解勿淡出，留给 keypoints
```

### 关键点拨

```python
self.play_keypoints_only(
    mp["keyPoints"], wait=6,
    diagram=diagram_all, from_scale=written_diagram_scale,
)
```

### 分段包裹

```python
with self.segment("draw-1", "1", "segments/03.mp4", "步骤说明"):
    ...
```

- `label`：画图段用 `"1"`、`"2"`；其他段用语义名（题型讲解、作答…）
- `gap_after=False`：通常用于 end（片尾后无需段间间隔）
- 完整版段间自动 `wait(3)`，不计入分段 mp4

### 题型讲解段（intro）一屏布局

片头与题型讲解合并为单段 `intro`，概念与特征**同屏上下分栏**；下板块相对概念块底边动态定位。详见 [lessons-learned.md §六](../../.cursor/skills/math-drawing-video/references/lessons-learned.md)。

## 数据读取

```python
data = self.load_problem()           # public/data/problems/{lessonNumber}.json
mp = self.main_problem               # mainProblem 字段
mp["keyPoints"]                      # 关键点拨文案
mp["originalQuestion"]               # 原题
```

`keyPoints` 过长时在 JSON 中用 `\n` 断句。
