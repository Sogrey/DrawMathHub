## 核心技巧（画图解题视频）

1. **结构固定**：intro（题型识别+要点）→ 题目 → 图解分步 → 列式 → 点拨 → 片尾  
2. **intro 一文案**：概念标题统一写 **「题型识别」**，不要「什么是 XX？」  
3. **图解按题型拆 Mixin**：`diagrams/xxx.py`，Scene 只编排动画  
4. **先 ql 后 qh**：`python scripts/render_all_videos.py --quality ql --lessons N`  
5. **标签错开**：Brace/文字上下或左右错位，双行图加大行距  
6. **点拨留图**：作答缩小图勿淡出，`play_keypoints_only(diagram=...)`  
7. **一题一型**：母题 JSON 勿混情境；视频只演母题

## 高频踩坑（必看）

| 坑 | 一句话处理 |
|----|------------|
| `set_opacity(0)` 再 `FadeIn` | 图永久消失 → 图形保持不透明再 FadeIn；文字用 `animate.set_opacity(1)` |
| PNG 塞进 `VGroup` | 改用 `Group` + `load_icon_png` |
| MathTex + `%` | 改用 `safe_text` |
| 标签叠字 | 总量与分段分上下；双行标签改右侧 |
| 折返画成「绕圈」 | 直角 `┐/└` + 方向箭头 |
| Pages 上 full.mp4 404 | manifest 写相对路径 `full.mp4` |
| 段间图丢了 | 下一段 `self.add` 显式保留 |

细节与代码样例 → Skill `lessons-learned.md` §十五、§一、§五。
